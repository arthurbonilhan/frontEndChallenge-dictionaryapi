## Iniciando a Aplicação
1. Rode `npm install` ou `yarn install`.<br />
2. Rode `yarn start` e acesse `http://localhost:3000`.<br />
3. Clique em uma palavra<br/>
4. Em caso de dúvidas, entre em contato.
